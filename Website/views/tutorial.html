Tutorial

1. Create an account. Sign Up!

2. Download the starter package

Click here to download the starter package for Linux.(version 0.3)(Latest Upload: 15:00hours, 14th October, 2011)

Click here to download the starter package for Windows.

3. Executing Sample bots in Linux

  1. Open terminal. Change your current directory to starterpackage/kkgame.
  2. To execute the sample bots(aa and bb) run ./game map1.txt aa bb.
    You can use any map present in starterpackage/maps.
  3. To see the game in action
    1. Change directory to starterpackage.
    2. Run ./view.
    3. Open viewer.html file in Mozilla Firefox or Google Chrome.

4.Executing the sample bots in Windows

  1. Open command prompt. Change the directory to starterpackage/kkgame.
  2. Run game.exe map1.txt aa bb
  3. Change the directory to starterpackage/viewer
  4. Run view.exe
  5. Open viewer.html file in Mozilla Firefox or Google Chrome.

5. Testing your bot in Linux

  1. Make a folder containing your bot programs and the makefile.
    1. Name the makefile as "makefile".
    2. The executable name should be "pacman".
    3. Name the folder same as bot name.
    4. Compress the folder. Use zip,tar or tar.gz. The compressed file name should also be same bot name.
    Example:Suppose your bot's name is "my_bot", then the makefile should create a executable called "pacman". The folder and the compressed folder name should be my_bot and my_bot.zip respectively.
  2. Place the compressed folder inside starterpackage/kkgame.
  3. Follow the same instructions as Executing Sample bots.
    In step 2, replace one of the bot names' with your bot name.
    Example: Run ./game map1.txt my_bot aa

6. Testing your bot in Windows

  1. Make a folder containing your bot programs.
    1. Name the folder with your bot name.
    2. Create an executable of your bot with the name "pacman" in this folder.
    3. Place the folder inside starterpackage/kkgame.
  2. Follow the same instructions as executing sample bots. In step 4 replace one of the bots with your bot name
    Example: game.exe map1.txt my_bot bb

7. Submit your code.The last date to upload or change bot is 23:59hrs,29 October 2011.
